yeast(Mfuzz)R Documentation

Gene expression data of the yeast cell cylce

Description

The data contains gene expression measurements for over 6000 genes of the yeast mutant cdc28 as performed and described by Spellmann et al. For details, see the reference.

Usage

data(sw)

Format

An object of class “marrayRaw”

Source

The data was downloaded from XXX and converted to an exprSet object.
